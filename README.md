## ğŸ”§ Installation & Setup

### Prerequisites
- Node.js (version 14 or higher)
- npm (comes with Node.js)

### Installation Steps

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd plot-to-palace-integrated
   ```

2. **Install dependencies**
   ```bash

3. **Start the development server**

```
